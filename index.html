Lua Loader Script

Copy this loader script. It handles authentication, HWID binding, and script loading automatically.

-- ScriptShield Loader v2.0 -- Place this in your executor local API_URL = "https://your-domain.vercel.app" local SCRIPT_KEY = "YOUR_KEY_HERE" -- Auto HWID local function getHWID() local hwid = game:GetService("RbxAnalyticsService"):GetClientId() if not hwid or hwid == "" then hwid = game:GetService("Players").LocalPlayer.UserId end return tostring(hwid) end -- HTTP Request local function request(data) local http = game:GetService("HttpService") local response = (syn and syn.request or http_request or request)({ Url = API_URL .. "/api/gateway", Method = "POST", Headers = { ["Content-Type"] = "application/json", }, Body = http:JSONEncode(data) }) return http:JSONDecode(response.Body) end -- Execute local function loadScript() local result = request({ action = "execute", key = SCRIPT_KEY, hwid = getHWID() }) if result.success and result.script then print("[ScriptShield] ✅ Authenticated | Version: " .. (result.meta and result.meta.version or "1.0")) print("[ScriptShield] ⚡ Load time: " .. (result.meta and result.meta.loadTimeMs or "?") .. "ms") loadstring(result.script)() else warn("[ScriptShield] ❌ " .. (result.error and result.error.message or "Unknown error")) end end loadScript()
API Documentation

Endpoints

POST /api/gateway — Main gateway (recommended)

POST /api/auth/validate — Key validation only

POST /api/scripts/loader — Script delivery (needs session)

POST /api/admin/keys — Key management

POST /api/admin/dashboard — Dashboard data

Quick Start

1. Set MASTER_KEY and ADMIN_KEY in Vercel Environment Variables

2. Store a script via the Scripts tab

3. Generate keys via the Keys tab

4. Give users the Lua loader with their key

Environment Variables

MASTER_KEY — Master encryption key (REQUIRED)

ADMIN_KEY — Admin access key (defaults to MASTER_KEY)

REQUIRE_SIGNATURES — Enable request signing

STRICT_UA_CHECK — Enable User-Agent checking
